> 日本の「大麻取締法」には国外犯規定があり、日本人が海外で大麻を所持・譲受（栽培を含む）等した場合、処罰の対象となる可能性があります。本プロジェクトは、法規制が整備されている国々における情報の提供と教育のみを目的としており、法に触れる行為を推奨するものではありません。
